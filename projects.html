Apache TinkerPop™

Apache TinkerPop is a graph computing framework for both graph databases (OLTP) and analytic systems (OLAP). Numerous vendors leverage TinkerPop and its graph traversal language Gremlin. RReduXis focused on the next major version of Apache TinkerPop — TinkerPop4. TinkerPop4 is making significant strides in applying distributed virtual machine, language, and stream theory to the graph computing space.

ReduX GMachine™

RReduXGMachine is a theoretical machine intended for universal distributed computing over any data structure, using any data processor via any data query language. The theoretical developments in GMachine provide insight into how to develop data-oriented products that are loosely coupled to underlying open source technologies.
